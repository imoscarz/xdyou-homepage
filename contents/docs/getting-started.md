# 欢迎使用XDYou

XDYou（西电You）是专为西安电子科技大学学生打造的综合性校园应用。

## 快速开始

### 下载安装

访问[下载页面](/#downloads)获取适合您设备的版本。

### 首次使用

1. 打开XDYou应用
2. 使用校园账号登录
3. 允许必要的权限（通知、位置等）
4. 开始使用各项功能

## 主要功能

### 📅 课程表

自动同步您的课程安排，支持：
- 周视图和日视图切换
- 课程提醒
- 教室查询

### 📝 考试查询

- 查看考试时间和地点
- 考前提醒
- 成绩查询

### 💳 校园卡

- 实时余额查询
- 消费记录
- 充值记录

### 📚 图书馆

- 图书搜索
- 借阅记录
- 在线续借
