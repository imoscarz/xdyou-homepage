# 常见问题

## 登录相关

### 无法登录怎么办？

1. 检查网络连接
2. 确认账号密码正确
3. 尝试重新登录

### 忘记密码怎么办？

请联系学校教务处重置密码。

## 功能使用

### 课程表不显示？

1. 检查是否已登录
2. 尝试手动刷新
3. 确认本学期已选课

### 考试信息不准确？

考试信息从教务系统同步，如有疑问请以教务系统为准。

## 其他问题

### 如何反馈问题？

- 在GitHub上提交Issue
- 发送邮件至support@xdyou.example.com

### 支持哪些平台？

目前支持：
- Android
- Windows
- Linux

iOS和macOS版本正在开发中。
